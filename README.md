# 🚦 Road Accident Analysis Dashboard

## 📌 Overview
Developed an interactive Road Accident Analysis Dashboard in Microsoft Excel to analyze accident trends, casualty statistics, and road safety performance.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## 📊 Dashboard Features
- Total Casualties
- Fatal, Serious & Slight Casualties
- Vehicle Type Analysis
- Road Type Analysis
- Road Surface Analysis
- Urban vs Rural Analysis
- Light Condition Analysis
- Monthly Accident Trends
- Interactive Filters

## 📈 Key Insights
- Analyzed casualty distribution by severity.
- Compared accidents across different vehicle types.
- Identified trends based on road conditions and locations.
- Generated actionable insights for road safety analysis.
